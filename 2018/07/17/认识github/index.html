认识github

github是什么

Do whatever you want.
——Chris（创始人之一）

　　于是，在2008年4月10号这一天，github正式成立。
　　github是运用git思想来工作的一个商业网站，所有开发者都可以在github上面为自己的项目创建一个主仓库。git是管理代码的工具，分布式是它的核心思想。

在github网站完成新用户注册

在github注册账户并登陆 。
sign up是注册，sign in是登陆

在github上完成新仓库的创建

点击创建仓库（仓库分公开的和私有的,公开的是免费的,私有的是收费的）。
弹出界面，第一行填仓库名,就叫个grjava,第二行是对这个仓库的描述,之后那个Public就是公共仓库的意思,接下来的README就是在仓库里创建一个README文件,可以往里写一些介绍你这个项目的功能之类的东西,再下面那个Add gitignore按钮,可以选择你这个项目是用什么语言之类的,后面那个License我没有选,点击”Create repository”

github应用

  1. 使用commit功能，标记存放源代码文件。
  2. 使用branch功能，实现对源代码进行版本控制。
  3. 使用fork功能，实现多人合作编写程序、小说、修订法规文献。
  4. wiki不会用
  5. issues不会用
  6. github新建静态网站。

应用github提高技能

修行之道：
关注大师的言行，
跟随大师的举动，
和大师一并修行，

  • 主动给大师的项目写wiki或提交测试用例，或者问题
